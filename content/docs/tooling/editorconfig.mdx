## `.editorconfig` – Editor-Level Formatting

### What is it?

`.editorconfig` is a shared config file that helps **maintain consistent coding styles** across different editors and IDEs.

### How it’s different from Prettier

| Feature           | `.editorconfig`                       | Prettier                                   |
| ----------------- | ------------------------------------- | ------------------------------------------ |
| **Scope**         | Whitespace, line endings, indentation | Full code formatting (quotes, semis, etc.) |
| **Tooling**       | Editor built-in plugins               | Code formatter tool                        |
| **Enforcement**   | Editor-based                          | CLI, pre-commit hooks                      |
| **Complex rules** | Not supported                         | Yes (e.g., trailing commas, quote style)   |

### Example `.editorconfig`

```ini
root = true

[*]
indent_style = space
indent_size = 2
end_of_line = lf
charset = utf-8
trim_trailing_whitespace = true
insert_final_newline = true

[*.md]
trim_trailing_whitespace = false
```

### What it does:

- Ensures 2-space indentation
- Uses LF (`\n`) line endings
- Removes extra trailing spaces
- Enforces UTF-8 encoding
